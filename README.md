# 메타버스 미륵사지 타임머신(Metaverse_Mireuksaji_Timemachine)

Unity로 문화재 관람 콘텐츠를 개발하고 Maya로 미륵사지 석탑을 모델링하여 제작된 문화재 관람 목적의 메타버스 플랫폼입니다.

개발 기간 : 2021.09.24 ~ 21.12.10

개발 환경 : Unity 3.20f, Maya

빌드 플랫폼 : PC (Windows)

개발 인원 : 5명(기획 1, 개발 2, 모델링 2)

담당 업무 : Unity로 구현 가능한 전반적인 개발을 담당
- Unity Photon 클라우드 서버 구축 및 채팅 기능 구현

- Transform 및 Rigidbody를 활용한 캐릭터 이동 기능 구현

- Cinemachine을 활용한 카메라 기능 구현

- UGUI를 활용한 UI 디자인 구현

- 미륵사지 설화 스토리 시나리오 작성 및 스토리 연출 구현

- 전체 콘텐츠 기획 및 협의
